체인구조

핵심: 관절부(핀–부시)가 체인의 Pain을 만든다
구성요소
플레이트 + 롤러 + 부시 + 핀
링크(Link)를 만들고, 링크가 다수 연결된 구조
Pain Point
관절부(핀–부시) = 소음·마모·정비
벨트(치형+코드)로 전환 시 Pain 제거 가능
핵심가치: 무윤활 · 저소음 · 유지보수 절감 · TCO(총소유비용) 전환
전환판단: 가치(고객문제) + 적용조건(설계/환경) + 리스크(안전/규제)

최종 방향

체인시장 → 벨트시장 전환, 경쟁사 접근시장 → DRB 접근

1) 체인시장 → 벨트시장 전환

체인의 고질 Pain(윤활·소음·정비 부담)을 벨트로 제거하는 전환 시장을 만든다

2) 경쟁사 접근시장 → DRB 접근

경쟁사가 이미 접근하는 체인 대체 구간 중 성공확률이 높은 구간을 DRB가 재정의해 침투
